# ASTRALMESS - Автономная система экзаменационных билетов

Эта версия приложения полностью автономна. Она не использует Firebase, Google Fonts или любые другие внешние сервисы. Все данные хранятся локально в файле `db.json`.

## Как запустить на своем сервере/компьютере

Если вы скачали проект и видите ошибку `"vite" не является внутренней или внешней командой`, следуйте этой инструкции.

### 1. Установка окружения
Установите [Node.js](https://nodejs.org/) (версия 18 или выше). Это основной движок для запуска сайта.

### 2. Установка всех библиотек
Откройте папку с проектом в терминале (командной строке) и введите:
```bash
npm install
```
**Это обязательный шаг.** Он скачает все необходимые файлы (Vite, Express, React) в папку `node_modules`. Без этого команды `build` и `start` работать не будут.

### 3. Сборка проекта (Build)
Чтобы сайт работал быстро и стабильно, его нужно "собрать":
```bash
npm run build
```
После этого появится папка `dist` — это готовый сайт.

### 4. Запуск (Start)
Теперь можно запускать сервер:
```bash
npm start
```
Сайт будет доступен по адресу `http://localhost:3000`.

## Как это работает
- **База данных:** Все билеты и брони (assignments) сохраняются в файл `db.json` в корне папки. Вы можете скопировать этот файл для резервной копии.
- **Бронирование:** В панели администратора можно указать фамилию студента и номер билета. Когда этот студент войдет и нажмет "Получить билет", он получит именно его. Для других студентов этот билет будет полностью исключен из выдачи (заблокирован).

  - Секретный код настроек: `777` (нажать 5 раз на логотип).

## Решение проблем
- **Ошибка "vite не найден":** Вы забыли выполнить `npm install`.
- **Ошибка "порт занят":** Закройте другие программы, использующие порт 3000, или измените порт в `server.ts`.
- **Данные пропали:** Проверьте наличие файла `db.json`. Если его удалить, база создастся заново пустой.

## Деплой на хостинг (SpaceWeb / VPS)
1. Убедитесь, что ваш тариф поддерживает **Node.js**.
2. Загрузите файлы проекта на сервер (кроме папки `node_modules`).
3. В терминале хостинга выполните `npm install` и `npm run build`.
4. В настройках Node.js на хостинге укажите файл запуска: `dist/server.cjs`.
5. Если хостинг требует другой порт, приложение автоматически его подхватит из системных настроек.
